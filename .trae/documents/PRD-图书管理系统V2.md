# 图书管理系统 - 产品需求文档 (PRD)

## 1. 产品概述

一个功能完善的个人图书管理系统，为用户提供完整的书籍管理体验。用户可以通过邮箱或手机号注册登录，拥有个人专属的书架和书籍分类管理功能。

**核心价值**：
- 安全可靠的用户账户系统
- 个性化的书架管理
- 灵活的书籍分类
- 美观舒适的视觉体验

## 2. 核心功能

### 2.1 用户系统

**用户角色**：
- 游客：未登录状态，只能浏览登录页
- 注册用户：已注册但未登录
- 登录用户：已登录的正式用户

**注册功能**：
- 支持邮箱注册
- 支持手机号注册
- 密码设置（至少6位）
- 确认密码

**登录功能**：
- 支持邮箱登录
- 支持手机号登录
- 记住登录状态
- 密码找回功能（简化版：显示提示）

### 2.2 个人主页

**用户信息展示**：
- 用户头像（默认头像）
- 用户名
- 邮箱/手机号
- 注册时间

**个人数据统计**：
- 图书总数
- 在读书籍数
- 已读书籍数
- 未读书籍数

**快捷操作**：
- 快速添加图书
- 最近阅读
- 我的书架入口

### 2.3 书架管理

**书架功能**：
- 创建书架（自定义名称）
- 编辑书架名称
- 删除书架（需确认）
- 设置书架封面颜色
- 默认书架：全部图书、正在阅读、已读完、想读

**书架操作**：
- 将图书添加到书架
- 从书架移除图书
- 移动图书到其他书架
- 书架排序

### 2.4 书籍分类

**系统预设分类**：
- 技术
- 文学
- 历史
- 经济
- 设计
- 哲学
- 科学
- 其他

**自定义分类**：
- 添加自定义分类
- 编辑分类名称
- 删除分类（图书移至"其他"）
- 分类排序

### 2.5 图书管理

**图书信息**：
- 书名（必填）
- 作者（必填）
- ISBN
- 出版社
- 出版日期
- 分类
- 简介
- 封面图片
- 评分（1-5星）
- 阅读状态（未读/在读/已读）

**图书操作**：
- 添加图书
- 编辑图书
- 删除图书
- 移动到书架
- 设置阅读状态
- 评分

### 2.6 搜索和筛选

**搜索功能**：
- 搜索书名
- 搜索作者
- 搜索ISBN
- 搜索出版社

**筛选功能**：
- 按书架筛选
- 按分类筛选
- 按阅读状态筛选
- 按评分筛选

## 3. 用户界面设计

### 3.1 页面结构

**登录/注册页**：
- Logo和标题
- 邮箱/手机号输入框
- 密码输入框
- 验证码（可选）
- 登录/注册按钮
- 切换登录/注册链接

**个人主页**：
- 顶部用户信息栏
- 统计卡片区
- 快捷操作区
- 最近阅读区

**书架管理页**：
- 书架列表
- 添加书架按钮
- 书架编辑功能

**图书列表页**：
- 顶部导航栏
- 搜索框
- 筛选标签
- 图书卡片网格
- 分类标签

**图书详情页**：
- 图书封面大图
- 详细信息展示
- 操作按钮

### 3.2 设计风格

**色彩方案**：
- 主色调：深蓝 `#1e40af`
- 强调色：橙色 `#f97316`
- 背景色：浅灰 `#f8fafc`
- 文字色：深灰 `#1f2937`
- 成功色：绿色 `#22c55e`
- 警告色：红色 `#ef4444`

**布局风格**：
- 卡片式设计
- 清晰的导航结构
- 响应式布局
- 流畅的动画效果

## 5. 技术实现

### 5.1 技术栈

- **前端**：React 18 + TypeScript + Vite
- **样式**：Tailwind CSS
- **图标**：Lucide React
- **数据**：LocalStorage（模拟后端）
- **状态管理**：React Context + useReducer
